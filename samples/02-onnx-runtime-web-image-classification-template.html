ONNX Runtime Web Image Classification Template

This beginner template assumes the teacher provides one small image-classification ONNX model, its input size, and a label list. A small MobileNet or SqueezeNet ONNX model is a good first classroom choice.

Before students edit anything:
  • Use one tested teacher-provided .onnx model.
  • Confirm the expected input tensor shape, for example [1, 3, 224, 224].
  • Keep the first goal simple: load the model and print raw outputs.
